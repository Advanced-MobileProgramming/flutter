# soobook

A new Flutter project.
책 정보 출처
이 프로젝트에서 사용된 책 정보는 다음 출판사들의 허락을 받아 사용되었습니다:

김영사, 비룡소, 위즈덤하우스, 한겨레출판, 다산책방, 새움.

책 정보 및 표지는 각 출판사와 저작권자의 소유이며, 무단 복제 및 배포를 금지합니다.
